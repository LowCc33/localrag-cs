# 部署指南

## 环境要求

### 最低配置
- CPU: 4核8线程以上
- 内存: 16GB以上
- 显卡: NVIDIA GPU，显存8GB以上（推荐12GB）
- 磁盘: 20GB以上空余空间

- CPU: 8核16线程
- 内存: 32GB
- 显卡: NVIDIA RTX 3090/4090 24GB显存
- 磁盘: 50GB SSD

# 脚本会自动启动三个模型服务，端口分别是：

## 第四步：启动服务

### 方式一：一键启动（推荐）
```bash

```

### 方式二：分步启动
```bash

/home/zbs/llama_serve.sh

# 2. 等待15秒，确认模型都起来

```

## 生产部署建议

### 1. 配置 systemd 服务
```bash
# 创建服务文件
sudo vi /etc/systemd/system/cs-api.service
```

```ini
[Unit]
Description=智能客服API服务
After=network.target

[Service]
Type=simple
User=zbs
WorkingDirectory=/home/zbs/smart-customer-service
ExecStart=/home/zbs/smart-customer-service/venv/bin/uvicorn app:app --host 0.0.0.0 --port 8000
Restart=always
RestartSec=5
StandardOutput=file:/home/zbs/smart-customer-service/logs/api.log
StandardError=file:/home/zbs/smart-customer-service/logs/api-error.log

[Install]
WantedBy=multi-user.target
```

```bash
# 启用并启动服务
sudo systemctl daemon-reload
sudo systemctl enable cs-api
sudo systemctl start cs-api
```

### 2. 配置日志轮转
```bash
# 配置 logrotate 防止日志太大
sudo vi /etc/logrotate.d/cs-service
```

```logrotate
/home/zbs/smart-customer-service/logs/*.log {
    daily
    rotate 30
    compress
    missingok
    notifempty
    create 0644 zbs zbs
}
```

### 3. 配置防火墙
```bash
# 只开放必要端口
sudo ufw allow 8000/tcp
sudo ufw reload
```

## 升级指南

### 升级代码版本
```bash
# 1. 停止服务
bash scripts/stop_all.sh

# 2. 备份当前版本
cp -r smart-customer-service smart-customer-service.backup

# 3. 覆盖新代码
unzip smart-customer-service-new.zip

# 4. 对比配置文件
diff config.py config.py.new

# 5. 重启服务
bash scripts/start_all.sh
```

### 升级模型版本
```bash
# 1. 下载新模型到 /home/zbs/models/

# 2. 修改 /home/zbs/llama_serve.sh 中的模型文件名

# 3. 重启服务
bash scripts/stop_all.sh
bash scripts/start_all.sh
```
